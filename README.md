# NumPy Learning Notes (Chinese)

这是一个面向初学者到进阶用户的 NumPy 学习开源仓库，包含：

- `NumPy学习指南.md`：可运行示例、分层练习、解析、常见坑点
- `NumPy与Matplotlib绘图完全教程.py`：NumPy + Matplotlib 系统教程、可运行 demo、项目化实战

## 快速开始

```bash
python "NumPy与Matplotlib绘图完全教程.py"
```

建议先阅读 `NumPy学习指南.md`，再按脚本中的 demo 逐个运行。

## 环境

- Python 3.10+
- numpy
- matplotlib

安装依赖：

```bash
pip install numpy matplotlib
```
